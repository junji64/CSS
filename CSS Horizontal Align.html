In CSS, several properties can be used to align elements horizontally.

Block 요소 정렬

블록 요소는 사용 가능한 전체 폭을 차지하며, 전과 후에 줄 바꿈이 있는 요소입니다.

Examples of block elements:

  • <h1>
  • <p>
  • <div>

For aligning text, see the CSS Text chapter.

이 장에서는 수평 레이아웃을 위해 블록 요소를 정렬하는 방법을 보여줍니다.

margin 속성을 사용한 중앙 정렬

블록 요소 left 과 right margins 을 "auto"으로 설정하여 중앙 정렬 할 수 있습니다.

Note: Using margin:auto; will not work in IE8 and earlier, unless a !DOCTYPE is declared.

left 과 right margins 을 auto 로 설정하면 사용할 수 있는 margin 을 동일하게 분할합니다. 결과로 요소는 중심에 놓입니다 :

Example

.center {
    margin: auto;
    width: 60%;
    border: 3px solid #73AD21;
    padding: 10px;
}
Try it yourself »

Tip: width 가 100% 이면 중앙정렬은 효과가 없다.

position 속성을 이용한 왼쪽 과 오른쪽 정렬

요소를 정렬하는 방법 중 하나는 절대 위치(absolute positioning)를 사용하는 것입니다 :

Example

.right {
    position: absolute;
    right: 0px;
    width: 300px;
    border: 3px solid #73AD21;
    padding: 10px;
}
Try it yourself »

Note: 절대 위치(Absolute positioned)의 요소들은 기본적인 흐름으로부터 제거되고, 요소들에 중첩 될 수 있다.

브라우저 간의(Crossbrowser) 호환성 문제

이렇게 요소를 정렬 할 때, <body> 요소에 대한 margin 과 padding 을 미리 정의하는 것이 좋습니다. 이것은 다른 브라우저들 있어서 시각적 차이를 방지합니다.

position 속성을 사용할 때, IE8 이전 버젼에는 문제가 있다. 컨테이너 요소가 (우리의 경우 <div class="container"> ) 지정된 폭을 가지고 있으며, !DOCTYPE 선언이 없는 경우, IE8 이전 버전은 오른쪽에 17px의 여백을 추가합니다. 이는 스크롤바을 위해 예약된 공간으로 보인다. position속성을 사용할 때는 항상 !DOCTYPE 선언을 설정해야한다.

Example

body {
    margin: 0;
    padding: 0;
}

.container {
    position: relative;
    width: 100%;
}

.right {
    position: absolute;
    right: 0px;
    width: 300px;
    background-color: #b0e0e6;
}
Try it yourself »

float 속성을 사용한 왼쪽과 오른쪽 정렬

요소를 정렬하는 하나의 방법은 float 속성을 사용하는 것이다:

Example

.right {
    float: right;
    width: 300px;
    border: 3px solid #73AD21;
    padding: 10px;
}
